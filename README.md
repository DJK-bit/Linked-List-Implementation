# Linked List Implementation

## 1. Project Description
This project enables the user to manually create and manipulate a linked list. Furthermore, the project enables the user to have a deeper understanding of linked lists without using Java's built in class.

## 2. Tech Stack/ Language
- Language: Java 
- IDE: Visual Studio Code

## 3. File Structure
- Main.java
- MyLinkedList.java
- ListNode.java
- README.md

## 4. Features
- add(ListNode head, int value): Adds a new node to the linked list.
- size(ListNode head): Returns the total number of nodes in the linked list.
- print(ListNode head): Prints all values in the linked list.
- get(ListNode head, int index): Returns the value at the given index in the linked list.
- remove(ListNode head, int index): Removes a node from the linked list with the given index.
- removeByValue(ListNode head, int value): Removes the node with the given value.
- insertAtIndex(ListNode head, int index, int value): Inserts a value at the specified index.

## 5. How to Run
javac *.java
java Main

## 6. Time Complexity
- add(ListNode head, int value): O(n)
- size(ListNode head): O(n)
- print(ListNode head): O(n)
- get(ListNode head, int index): O(n) 
- remove(ListNode head, int index): O(n)
- removeByValue(ListNode head, int value): O(n)
- insertAtIndex(ListNode head, int index, int value):O(n)
